* OBS: Essa implementação de String armazena sempre um caractere ao final extra, que é o caractere '\0' como é na linguagem C. Logo, mesmo ao criar uma string com string_new_empty() por exemplo (que, na prática, chama a função string_new_with_size(0)), sempre será alocado uma unidade de espaço a mais (neste caso 1) para conseguir adicionar o caractere '\0'. Então, sempre ao final de qualquer uso de string_new_empty() (ou de qualquer função e inicializador), libere a memória com o string_free().
